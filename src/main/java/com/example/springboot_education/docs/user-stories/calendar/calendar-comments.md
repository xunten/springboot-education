## 9. Thời khóa biểu

**User Story**  
Là học sinh, tôi muốn xem thời khóa biểu lớp học để theo dõi lịch học của mình.

**Acceptance Criteria**

- Xem thời khóa biểu theo ngày/tuần.
- Giáo viên có thể tạo hoặc cập nhật lịch học.
- Đồng bộ lịch giữa học sinh và giáo viên.

**Priority:** Low  
**Story Points:** 3
